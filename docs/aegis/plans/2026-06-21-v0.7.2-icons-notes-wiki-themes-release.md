# Architecture

Renderer remains React + CSS. Generated image assets are stored under `assets/` and imported through small owner modules. Electron main process keeps its theme ID whitelist in `electron/themeIds.cjs`.

# Tech Stack

- Electron main process: `electron/main.cjs`, `electron/themeIds.cjs`
- React UI: `src/App.tsx`, `src/components/NotesView.tsx`
- Theme definitions: `src/core/themes.ts`
- Page topbar/image assets: `src/core/pageTopbars.ts`, new page icon owner
- CSS: `src/styles.css`, `src/styles/components.css`
- Tests: Vitest
