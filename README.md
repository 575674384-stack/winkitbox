# WinKitBox

WinKitBox 是一个自用 Windows 装机工具箱。目标是换电脑后不用再一个个找下载页，而是导入自己的方案，一键恢复常用软件环境。

## v0.1 范围

- 工具目录：开源优先、官方源优先，内置常见 Windows 工具。
- 真实图标：优先展示软件官网图标或 GitHub 项目头像。
- 装机方案：新电脑必装、开源优先、卸载清理、桌面整理。
- 安装计划：根据勾选项生成 winget、直链下载、便携包解压和安装器执行步骤。
- 自定义目录：可设置 WinKitBox 管理的工具目录，便携工具、安装器缓存和解压依赖都会放在这里。
- 安装后打开：工具卡片内提供打开入口，桌面版会尝试从开始菜单、常见命令或管理目录启动。
- 卸载清理：已安装工具支持单个卸载和勾选后批量卸载。
- GitHub 榜单：内置周榜 / 月榜阅览、语言筛选、中文自动翻译、候选收藏和 clone 命令复制。
- GitHub 网络：桌面版支持系统代理、直连和手动代理，可填写 GitHub Token 提升 API 限额。
- 更新检查：桌面版可检查 GitHub 最新发行版，并打开下载页获取安装版或便携版。
- 桌面体验：最小化/关闭默认收纳到系统托盘，可从托盘恢复或退出。
- 操作日志：预览脚本、模拟运行、用户主动执行 PowerShell 安装/卸载计划。
