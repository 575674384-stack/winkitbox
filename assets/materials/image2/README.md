# WinKitBox Image2 素材库

这个目录只存放备用 UI 美化素材，当前未接入代码。

## 目录

- `banners/`: 顶部栏横幅素材，适合页面 `command-bar`、标题栏、设置卡片头图。
- `stickers/`: 单张 Q 版角色贴纸，适合空状态、小浮层、工具卡装饰。
- `empty-states/`: 空状态 / 成功 / 失败 / 加载类整图。`atlas-empty-states-with-copy.png` 带中文文案，适合直接作为空状态大图使用。
- `ui-sheets/`: 高密度 atlas 图集。每张图内包含多个小图标、徽章、状态图、卡片纹理或装饰件，适合后续按网格裁切。
- `derived/contact-sheet.png`: 当前素材总览预览图。
- `manifest.json`: 素材文件名、用途标题和文件大小清单。

## 使用建议

- atlas 类图集优先按固定网格裁切，避免直接整张铺在页面上。
- 带文字的空状态图适合整图展示，不建议裁切成小图标。
- 当前素材多为 PNG 位图，不是 SVG。需要 SVG 风格时建议后续单独用矢量工具或 CSS 图形重做。
- 这些素材还没有压缩和接入构建，真正使用前建议按目标尺寸二次导出，避免包体变大。
